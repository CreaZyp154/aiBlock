# aiBlock
uBlock filters to remove obnoxious AI features

## Add filter to uBlock Origin:
Open uBlock Origin settings and go to `Filter lists` tab, scroll all the way to the botton to `import` and paste this:
```
https://raw.githubusercontent.com/CreaZyp154/aiBlock/refs/heads/main/aiBlock.txt
```
